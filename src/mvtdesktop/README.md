Development notes
--

- Run the app in the developer mode:
```briefcase dev```

- Build the app:
```briefcase build```

- Run the app:
```briefcase run```

Remember to run 
```briefcase update```

before `run` if you've made changes to the app, to modify the code in the application bundle.

Alternatively run:
```briefcase run -u```

- Build the installer:
```briefcase package```

--
